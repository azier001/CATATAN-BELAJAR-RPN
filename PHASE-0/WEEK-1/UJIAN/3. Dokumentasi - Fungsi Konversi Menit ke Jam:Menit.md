<a name="apa-itu-template-literal"></a>
### ğŸ’¬ Apa itu Template Literal?

Template literal adalah cara modern menulis string menggunakan backtick (`` ` ``) yang bisa menyisipkan variabel dengan `${}`.

**Contoh:**
```javascript

console.log("Nama: " + nama + ", Umur: " + umur)

```

**Kegunaan dalam kode kita:**
```javascript

// Menggabungkan jam dan menit dengan format "jam:menit"
```

<a name="apa-itu-ternary-operator"></a>
### â“ Apa itu Ternary Operator?

Ternary operator adalah **if-else singkat** dengan format: `kondisi ? nilaiJikaTrue : nilaiJikaFalse`

**Contoh:**
```javascript
// Cara biasa (if-else)
let status
if (umur >= 18) {
  status = "Dewasa"
} else {
  status = "Anak-anak"
}

// Cara ternary (singkat)
let status = umur >= 18 ? "Dewasa" : "Anak-anak"
```

**Kegunaan dalam kode kita:**
```javascript
restTime < 10 ? `0${restTime}` : restTime
// Jika menit < 10, tambahkan "0" di depan
// Jika tidak, pakai apa adanya
```

**Penjelasan:**
- `restTime < 10` â†’ **kondisi**
- `` `0${restTime}` `` â†’ nilai jika **true** (contoh: "03", "05")
- `restTime` â†’ nilai jika **false** (contoh: 28, 53)
